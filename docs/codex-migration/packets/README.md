# Internal Packet Archives

This directory stores internal Substrate roadmap packet archives that are
not used as the default clone-and-own scaffold.

- `substrate-internal-upgrade-roadmap-wbs-2026-02-17.json`:
  historical internal upgrade roadmap (UPG-001..UPG-066) preserved for
  migration traceability.

Default template baseline for new clones is `.governance/wbs.json`.
