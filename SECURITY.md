# Security Policy

## Supported Versions

This repository follows the latest main branch for security updates.

## Reporting a Vulnerability

Please report vulnerabilities privately rather than opening public issues.

Recommended report contents:
- affected file(s)/component(s)
- reproduction steps
- impact assessment
- suggested remediation (if known)

Send reports via GitHub Security Advisories (preferred) or direct maintainer contact.

## Scope Notes

Governance and execution integrity paths are high priority:
- `.governance/wbs_cli.py`
- `.governance/wbs_server.py`
- `src/governed_platform/`
- state/schema governance files under `.governance/`

## Disclosure

After fix validation, coordinated disclosure is preferred.
